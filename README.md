# baidupcs-go

[![Build](https://github.com/gsmlg-ci/baidupcs-go/actions/workflows/build.yml/badge.svg)](https://github.com/gsmlg-ci/baidupcs-go/actions/workflows/build.yml)

Docker image published to:
- `docker.io/gsmlg/baidupcs-go`
- `ghcr.io/gsmlg-dev/baidupcs-go`

## Usage

```bash
# From Docker Hub
docker pull gsmlg/baidupcs-go:latest

# From GitHub Container Registry
docker pull ghcr.io/gsmlg-dev/baidupcs-go:latest
```

## Build

```bash
docker build -t gsmlg/baidupcs-go .
```
